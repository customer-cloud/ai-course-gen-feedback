## 🎯 このリポジトリについて

**AI Course Content Generator**は、Google Gemini AIを活用した教育コンテンツ制作プラットフォームです。このリポジトリは、ユーザーの皆様からのフィードバックを収集し、プロダクトの改善に役立てるために設けられています。

### 主な機能
- 🤖 **AI駆動コース生成**: Gemini AIによる高品質な教育コンテンツ自動生成
- 📊 **ダッシュボード**: コース管理・進捗追跡
- 🎨 **カスタマイズ**: 業界別テンプレート対応
- 📱 **レスポンシブデザイン**: PC・タブレット・スマートフォン対応
- 🔐 **Firebase統合**: 認証・データベース・ホスティング

## 📋 Issue提出ガイドライン

### 良いIssueの書き方

#### バグ報告の場合
- **再現手順**を明確に記載
- **期待される動作**と**実際の動作**を説明
- **スクリーンショット**や**エラーメッセージ**を添付
- **環境情報**（OS、ブラウザ、バージョン）を記載

#### 機能リクエストの場合
- **解決したい課題**を説明
- **提案する機能**の具体的な内容
- **期待される効果**を記載
- **代替案**があれば併せて提案

#### 質問の場合
- **何を実現したいか**を明確に
- **試したこと**を記載
- **関連ドキュメント**のリンク（あれば）

## 📊 Issue対応プロセス

1. **Issue作成**: ユーザーがIssueを作成
2. **トリアージ**: 開発チームが優先度を判定（Severityラベル付与）
3. **対応**: 優先度に応じて対応
4. **フィードバック**: 進捗をIssueコメントで共有
5. **クローズ**: 解決後にIssueをクローズ

### 対応時間の目安
| Severity | 対応時間 |
|----------|---------|
| Sev.1 (Critical) | 即座 |
| Sev.2 (High) | 24時間以内 |
| Sev.3 (Medium) | 48時間以内 |
| Sev.4 (Low) | 1週間以内 |
| Sev.5 (Trivial) | 1ヶ月以内 |
